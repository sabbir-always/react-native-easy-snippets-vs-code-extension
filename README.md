# React Native Easy Snippets

A collection of useful React Native & Expo snippets for faster development.

## Features
- React Native View & Text boilerplates
- Screen templates: SafeAreaView, ScrollView
- Buttons, Inputs, Images
- Supports className / NativeWind styling

## Usage
1. Open a `.tsx` or `.jsx` file
2. Type the snippet prefix, for example:
   - `screenview` → Creates a basic screen with View & Text
   - `screensafe` → Screen wrapped in SafeAreaView
   - `screenscroll` → Scrollable screen using ScrollView
   - `screensafescroll` → Scrollable screen with SafeAreaView
   - `viewstyle` → View with style object
   - `viewclass` → View with className
   - `textstyle` → Text with style
   - `textclass` → Text with className
   - `btn` → Inline Button with alert
   - `pressableroute` → Pressable button that navigates to a route
   - `touchable` → TouchableOpacity button with alert
   - `inputtext` → Default text input
   - `inputemail` → Email input with autoCapitalize='none'
   - `inputnumber` → Numeric input
   - `inputphone` → Phone number input
   - `inputpassword` → Password input
   - `imgsrc` → Local image using require
   - `imgnet` → Network image using uri
   - `safearea` → SafeAreaView wrapper with styling
   - `scrollview` → ScrollView wrapper with padding and styling
3. Press `Enter` to expand
